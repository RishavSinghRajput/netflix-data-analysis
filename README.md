📊 Netflix Data Analysis using Python :-

📌 Overview

This project focuses on analyzing Netflix's dataset to uncover insights about content distribution, trends, and user preferences. Using Python and data analysis libraries, we perform data cleaning, exploration, and visualization to better understand Netflix's content strategy.

🎯 Objectives
Analyze the distribution of Movies vs TV Shows
Identify most popular genres
Study content growth over the years
Analyze country-wise content production
Understand trends in ratings and durations

🛠️ Technologies Used
Python
Pandas
NumPy
Matplotlib
Seaborn
Jupyter Notebook

📂 Dataset
Source: Netflix dataset (CSV format)
Contains information such as:
Title
Type (Movie/TV Show)
Director
Cast
Country
Release Year
Rating
Duration
Genre



🔍 Steps Performed
1. Data Cleaning
Handled missing values
Removed duplicates
Converted data types

2. Exploratory Data Analysis (EDA)
Distribution of Movies vs TV Shows
Top genres analysis
Content added over years
Country-wise content production

3. Data Visualization
Bar charts
Pie charts
Heatmaps
Trend analysis graphs


📈 Key Insights
Movies are more frequent than TV Shows on Netflix
Certain genres like Drama and Comedy dominate
Significant growth in content after 2015
The US produces the highest amount of content
